How did you sign up for Polymarket?

→ Email / Google / Apple — use the Email / Magic Account guide
→ MetaMask / WalletConnect / any web3 wallet — use the EOA Wallet guide

What you'll need before starting:

  • ✓ A funded Polymarket account (USDC on Polygon)
  • ✓ A private key for the bot to sign transactions
  • ✓ Your funder wallet address (if using email account)
  • ✓ Access to your bot's web dashboard

Private Key Security

Never share your private key with anyone. The bot only needs a key for a dedicated signing wallet — not your main holding wallet. Keep funds in a separate account.
